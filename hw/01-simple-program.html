Quiz 2: Programme simple

Dans cet exercice, tu seras amené à écrire un programme simple en Python permettant de calculer l'accéleration (positive ou négative) d'un objet.

Accéleration

Formule d'accélération: a = Δv / Δt = (vf - vi)/(tf - ti)
vf (Vitesse finale); vi (Vitesse initiale); tf (Temps final); ti (Temps initial)

Pour calculer l'accélération moyenne sur une période de temps, nous soustrayons la vitesse initiale de la vitesse finale, puis divisons le résultat par l'intervalle de temps.
Si la vitesse finale est inférieure à la vitesse initiale, l'accélération sera un nombre négatif ou la vitesse à laquelle l'objet ralentit.

Test

Une voiture accélère uniformément de 24,7 m/s à 36,1 m/s en 2,47 s.
Variables: vf = 46,1 m/s, vi = 18,5 m/s, tf = 2,47 s, ti = 0 s.

Une motoneige roulant à 12,4 m/s s'arrête en 2,55 s après avoir freiné.
Variables: vf = 0 m/s, vi = 22.,4 m/s, tf = 2,55 s, ti = 0 s.

Instructions

Écris un programme qui permet de calculer l'accélération (en utilisant la formule présentée) dans les deux cas de tests décrits plus haut.

  • si l'accélération est positive, le programme affiche "Accélération de: [valeur]"
  • si l'accélération est négative, le programme affiche "Décélération de: [valeur]"
  • la valeur affichée doit toujours être positive ou nulle.

Le code doit être complet, avec des commentaires qui indiquent le nom du fichier, l'auteur (ton nom), la date de création, une brève description de l'utilité du programme.
Il doit aussi y avoir des commentaires qui expliquent à quoi correspondent chaque variable et constante, et le fonctionnement du programme (seulement si c'est utile!)
